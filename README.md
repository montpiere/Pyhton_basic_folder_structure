One of a basic sample folder structure for Python projects.

### Folders:
#### config/
- configuration files (csv, py, json, ..) 
#### data/
- datafiles (databases, csv, txt, ..)
#### docs/
- documentaries (md, csv, txt, jpg, psd, ..)
#### src/
- all other python source codes, modules, scripts, ..
